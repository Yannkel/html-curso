Texto Marcado

Podemos, também, criar um texto marcado usando a tag mark.
A cor padrão da tag mark é amarelo, mas caso queira mudar apenas usar parâmetro |style="background-color: nome_da_cor ou código;"|.

Outro texto marcado: Lorem ipsum dolor sit amet consectetur adipisicing elit.

Mais outro: Lorem ipsum dolor sit amet, consectetur adipisicing elit.

Para mudar a cor global do mark, dentro da tag head adicionar a tag style, assim:
<style> { background-color: nome_da_cor ou código; } </style>
